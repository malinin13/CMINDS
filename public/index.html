Plugin Description

The WordPress Business Directory plugin by CreativeMinds allows you to build an embedded business directory on your WordPress site. This WordPress plugin makes it easy to start any kind of local directory and include it on your site.

Are you looking to build relationships in your local business community? Become a trusted resource of local listings and information by including the business directory plugin for WordPress and start building a database of local businesses.

  • Bonus: Add the CM Business Directory Community Add-On for only $29 to allow users to submit and edit their own business listings within the business directory.
  • Watch our online WordPress directory plugin tutorial.
  • View this plugin live on ListWP.com

Using the WordPress Directory Plugin

  • Build a local business directory on your WordPress site
  • Allow owners to add business listings and manage their local directory profiles
  • Users can filter local listing results according to pre-defined search parameters
